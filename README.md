# 🎟️ Virtual Event Manager

A full-stack web application for managing virtual events, built using React for the frontend and Node.js with Express for the backend. The platform enables users and administrators to manage event workflows efficiently through a structured client-server architecture.

---

## 🚀 Features

- 🔐 User & Admin Authentication
- 📅 Event Creation and Management
- 👥 Role-Based Access Control
- 🌐 RESTful API Integration
- ⚛️ Modular React Frontend
- 🖥️ Express.js Backend Server

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Ayushi227/Virtual-Event-Manager.git
cd Virtual-Event-Manager
```

---

### 2️⃣ Setup Frontend

```bash
cd frontend
npm install
npm start
```

The frontend will typically run on:
```
http://localhost:3000
```

---

### 3️⃣ Setup Backend

Open a new terminal:

```bash
cd backend
npm install
npx nodemon server.js
```

The backend server will run on:
```
http://localhost:5000
```

## 🧠 How It Works

- The React frontend communicates with the Express backend through REST APIs.
- Authentication manages role-based access (User/Admin).
- Backend handles event data processing and business logic.
- Frontend dynamically renders content based on user role.
